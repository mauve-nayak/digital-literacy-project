Project Overview

As a Student Digital Ambassador at VIT Bhopal University, I have developed this portfolio to guide my peers through the digital world—from creating professional online profiles to communicating safely and responsibly. This repository documents my work across five modules, demonstrating my ability to use essential digital tools and maintain a secure online presence.

This project is organized into specific folders for each task as per the project requirements:
